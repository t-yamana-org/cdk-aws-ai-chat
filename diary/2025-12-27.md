- WSL2 では使えている amazonq VSCode拡張が devcontainer では起動時にエラーになるようになっていた。
    - エラーメッセージ：
        - The Amazon Q Language Server server crashed 5 times in the last 3 minutes. The server will not be restarted.
    - 原因：内部的に呼び出している時のコード
        - [info] Command: (not started) [/home/node/.cache/aws/toolkits/language-servers/AmazonQ/1.53.0/servers/node --max-old-space-size=8196 /home/node/.cache/aws/toolkits/language-servers/AmazonQ/1.53.0/servers/aws-lsp-codewhisperer.js --nolazy --preserve-symlinks --stdio --pre-init-encryption --set-credentials-encryption-key]
        - 推奨メモリ使用量が増えていた
    - 解決法：
        - Docker Desktop 設定で使えるメモリ量を8GBに増やした
