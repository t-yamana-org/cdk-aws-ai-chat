- CDK開発用のNode.js+TSイメージの選択方法
    - .devcontainer.json を他からコピペで持ってくるのではなく
    - 左下から [Add Devcontainer Configuration files] で node.js と検索
    - 最新のLTSイメージをリストアップしてくれるので古い記事からコピペするより良い
- GitHubレポジトリの作成
    - リモートレポジトリを作成するときにREADME,ライセンス,ignoreなどを追加しないのがポイント
    - 完全にブランクで作成した時だけローカルからのプッシュ案内が表示される。
- profile はホストからマウントすると configure する手間を省ける
    - ホストOSとユーザーが変わるので config の権限に注意しないと読めなくなる
    - aws sso login --profile admin から作業する
